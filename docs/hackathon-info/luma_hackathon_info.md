## Available APIs & Integrations
The hackathon features several powerful integrations for building agentic tools:
*   **AWS:** Cloud infrastructure and AI models (Bedrock).
*   **Vapi:** Voice infrastructure for AI agents.
*   **WunderGraph:** API orchestration.
*   **Tinyfish:** Infrastructure for AI Web Agents.
*   **Guild.ai:** Code-first, model-agnostic platform.
*   **Nexla:** Data integration and transformation.
*   **Redis:** Real-time data layer.
*   **TigerData (Ghost):** Research data infrastructure.
*   **Akash:** Decentralized GPU compute.
*   **Chainguard:** Secure container images.
*   **InsForge:** Backend platform for AI-native builders.
