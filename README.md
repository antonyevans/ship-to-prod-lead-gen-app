# Lead Gen Agent

Welcome to the repo for the Lead Gen Agent built for the Ship to Prod Hackathon.

## Structure
* **/docs**: Contains all documentation
  * **/planning**: Team execution plans and structure (`TEAM_PLAN.md`).
  * **/hackathon-info**: Transcripts and OCR from the rules/Luma events.
  * **/assets**: Images and sponsor slides.
* **/scripts**: Utility scripts (like our custom OCR tool).
* **/src**: Main source code for our web agent, backend, and Vapi connections.
